I tried to find all the ways to style custom list bullets with no extra elements and while correctly lining up wrapped lines. There are quite a few. Can you find more?

One thing to take into account is the fact that we want to avoid the typical float issue where wrapped list items do not match up. Take a look at every 3d item, it should line up correctly..

Disclaimer: some of these solutions are not very practical. Use something robust and simple in real code ;-)
